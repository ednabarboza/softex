1- Do ponto de vista do protocolo cliente-servidor a web funciona através de requisições. O cliente envia uma solicitação ao servidor de uma página acessada, através da URL digitada, por exemplo. O servidor recebe essa solicitação e envia a resposta também através do protocolo HTTP, essa resposta através de um código indica o status da solicitação (pode ser por exemplo um código 200 que indica que está tudo ok, ou um código 401 que indica não autorizado, etc). Nesse transporte de informações o protocolo TCP/IP é usado para reduzir o tamanho das informações e transportar as mesmas em pequenos pacotes em sequências binárias por meio de fios elétricos, fibras óticas e redes sem fio. 2 - Para criação de um projeto em Node.js (com a pasta já criada, clicamos nela e abrimos o terminal): Passo 1: npm init (criação do package.json) Passo 2: npm install + nome_da_biblioteca_que_quer_instalar
